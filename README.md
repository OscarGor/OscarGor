# 阴盘奇门足球AI预测系统

## 项目概述
基于阴盘奇门遁甲的足球比赛预测系统，结合AI参数优化和自动更新机制。系统通过分析奇门格局，应用V5.2参数体系，进行比赛结果预测和技术指标分析。

## 快速开始

### 1. 环境要求
- 现代浏览器（Chrome 80+, Firefox 75+, Safari 13+）
- GitHub账号
- Supabase账号

### 2. 一键部署
[![部署到GitHub Pages](https://img.shields.io/badge/部署-GitHub_Pages-blue)](https://github.com/new?template_name=yinpan-qimen-football)

### 3. 本地开发
```bash
# 克隆仓库
git clone https://github.com/[你的用户名]/yinpan-qimen-football.git
cd yinpan-qimen-football

# 安装本地服务器（可选）
npm install -g live-server
